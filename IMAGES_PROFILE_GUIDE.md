# Guide des Images pour la Page Profil

## Images à ajouter dans `src/assets/images/`

### 1. **profile-hero-illustration.png** (Bannière principale)
- **Emplacement** : Bannière de bienvenue en haut de page
- **Dimensions recommandées** : 400x300px
- **Style** : Illustration moderne d'une personne travaillant sur ordinateur (comme votre exemple)
- **Couleurs** : Utiliser les tons #113F67, #84cabe, #e5c3c9
- **Suggestions** :
  - Personne avec ordinateur portable
  - Style flat design ou illustration vectorielle
  - Ambiance positive et motivante
  - Fond transparent ou avec éléments décoratifs

### 2. **imc-illustration.jpg** (Carte IMC)
- **Emplacement** : En-tête de la carte IMC (résultats)
- **Dimensions recommandées** : 600x180px
- **Style** : Image liée à la santé, fitness, balance
- **Suggestions** :
  - Balance ou pèse-personne moderne
  - Personne en tenue de sport
  - Graphiques de santé
  - Ruban à mesurer
  - Couleurs dominantes : turquoise (#84cabe)

### 3. **calories-illustration.jpg** (Carte Calories)
- **Emplacement** : En-tête de la carte Besoins Caloriques
- **Dimensions recommandées** : 600x180px
- **Style** : Image liée à la nutrition, énergie
- **Suggestions** :
  - Aliments sains et colorés
  - Assiette équilibrée
  - Fruits et légumes
  - Symbole d'énergie/éclair
  - Couleurs dominantes : rose pâle (#e5c3c9)

### 4. **fitness-motivation.jpg** (Section motivation)
- **Emplacement** : Carte de motivation en bas de page
- **Dimensions recommandées** : 400x300px
- **Style** : Image inspirante de fitness/sport
- **Suggestions** :
  - Personne en plein exercice
  - Yoga ou méditation
  - Course à pied
  - Salle de sport moderne
  - Ambiance énergique et positive

## Sources d'images gratuites recommandées

1. **Undraw.co** - Illustrations personnalisables (parfait pour hero-illustration)
   - https://undraw.co/illustrations
   - Vous pouvez changer les couleurs pour matcher votre palette

2. **Freepik** - Images et illustrations
   - https://www.freepik.com/
   - Recherchez : "fitness illustration", "health dashboard", "nutrition"

3. **Unsplash** - Photos haute qualité
   - https://unsplash.com/
   - Recherchez : "fitness", "healthy food", "workout"

4. **Pexels** - Photos gratuites
   - https://www.pexels.com/
   - Recherchez : "health", "nutrition", "exercise"

5. **Storyset by Freepik** - Illustrations animées
   - https://storyset.com/
   - Catégories : Health, Fitness, Food

## Palette de couleurs à utiliser

- **Bleu foncé** : #113F67 (textes, éléments principaux)
- **Turquoise** : #84cabe (accents, boutons, IMC)
- **Rose pâle** : #e5c3c9 (accents secondaires, calories)

## Instructions d'installation

1. Téléchargez les images
2. Placez-les dans le dossier : `src/assets/images/`
3. Renommez-les exactement comme indiqué ci-dessus
4. Redémarrez le serveur Angular si nécessaire

## Exemple de structure de dossier

```
src/
└── assets/
    └── images/
        ├── profile-hero-illustration.png
        ├── imc-illustration.jpg
        ├── calories-illustration.jpg
        └── fitness-motivation.jpg
```

## Notes importantes

- Optimisez les images (compression) pour de meilleures performances
- Format recommandé : PNG pour les illustrations, JPG pour les photos
- Taille maximale : 200-300 KB par image
- Utilisez des images libres de droits
